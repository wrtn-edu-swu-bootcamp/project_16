## 🎯 핵심 컨셉

**마이크로러닝 기반 실시간 언어 학습 도구**

X에서 외국어 트윗을 읽을 때, 주요 단어를 자동 추출하고 뜻·발음·예문을 제공하여 3-10분 단위로 자연스럽게 학습

## 🔧 기술 스택 (확정)

- **AI/NLP**: Google Gemini 2.5 Pro (품사 태깅, 단어 추출)

- **사전**: Free Dictionary API
- **자동화**: n8n (워크플로우 엔진)
